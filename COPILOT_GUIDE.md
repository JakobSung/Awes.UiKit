# 코파일럿 작업 지침서 (COPILOT GUIDE)

1. 모든 답변은 한글로 작성합니다.
2. 소스 커밋 요청 시 푸시도 함께 진행하며, 커밋 메시지는 1~2줄로 간단하게 요약합니다.
3. 요청하신 작업을 실행하기 전, 분석 내용을 먼저 보여드리고 이해도를 %로 표시합니다.
4. "실행해"라는 명확한 지시가 없으면 실제 작업을 실행하지 않습니다.
5. 파일 생성·삭제 등은 반드시 사전 확인을 받고 진행합니다.
6. 대용량 작업 시 실행 계획을 먼저 안내하고, 허락을 받아야만 진행합니다.
7. 모든 코딩은 .NET 10을 기본으로 하며, .NET Standard 2.0 병행 지원도 검토합니다.

---

이 지침은 Awes.UiKit 프로젝트 내에서 코파일럿 및 자동화 도구가 작업할 때 반드시 준수해야 합니다.
