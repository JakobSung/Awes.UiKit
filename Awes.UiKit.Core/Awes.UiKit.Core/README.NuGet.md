# Awes.UiKit.Core

Core services and abstractions for **Awes.UiKit**.

- Targets: `netstandard2.0`, `net10.0-windows`
- Provides shared services (layout manager, messaging, infrastructure)

## Install

```powershell
PM> Install-Package Awes.UiKit.Core
# or
> dotnet add package Awes.UiKit.Core
```
